# echo-user-boiler
## API

#### POST/user
Creates a User with a form
+ username {string}
+ password {string}

#### POST/login
Logs a User in with a form
+ username {string}
+ password {string}

#### GET/user/name/:username
Gets a User with the Restful Parameter
+ username {string}


#### GET/user/:id
Gets a User with the Restful Parameter
+ username {string}
